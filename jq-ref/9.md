# jQuery 事件 - die() 方法



## 实例

移除所有通过 live() 方法向 p 元素添加的事件处理程序：

```
$("p").die();

```

## 定义和用法

die() 方法移除所有通过 live() 方法向指定元素添加的一个或多个事件处理程序。

### 语法

```
$(_selector_).die(_event_,_function_)
```

| 参数 | 描述 |
| --- | --- |
| _event_ | 必需。规定要移除的一个或多个事件处理程序。由空格分隔多个事件值。必须是有效的事件。 |
| _function_ | 可选。规定要移除的特定函数。 |
