# jQuery UI API - .scrollParent()

## 所属类别

[方法（Methods）](ref-methods.html) | [UI 核心（UI Core）](ref-ui-core.html)

## 用法

**描述：**获取最近的可滚动的祖先。

**返回：**[jQuery](//api.jquery.com/Types/#jQuery)

```
.scrollParent()
```

该方法不接受任何参数。

该方法查找最近的可滚动的祖先。换句话说，`.scrollParent()` 查找当前所选元素在其内滚动的元素。

该方法只在包含一个元素的 jQuery 对象上工作。
