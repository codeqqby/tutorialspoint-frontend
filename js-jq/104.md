
# ECMAScript 赋值运算符




## 赋值运算符

简单的赋值运算符由等号（=）实现，只是把等号右边的值赋予等号左边的变量。

例如：

```
var iNum = 10;
```

复合赋值运算是由乘性运算符、加性运算符或位移运算符加等号（=）实现的。这些赋值运算符是下列这些常见情况的缩写形式：

```
var iNum = 10;
iNum = iNum + 10；

```

可以用一个复合赋值运算符改写第二行代码：

```
var iNum = 10;
iNum += 10；

```

每种主要的算术运算以及其他几个运算都有复合赋值运算符：

*   乘法/赋值（`*=`）
*   除法/赋值（`/=`）
*   取模/赋值（`%=`）
*   加法/赋值（`+=`）
*   减法/赋值（`-=`）
*   左移/赋值（`<<=`）
*   有符号右移/赋值（`>>=`）
*   无符号右移/赋值（`>>>=`）
