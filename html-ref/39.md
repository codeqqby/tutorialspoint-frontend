# HTML &lt;dt&gt; 标签

## 实例

```
<dl>
   <dt>计算机</dt>
   <dd>用来计算的仪器 ... ...</dd>
   <dt>显示器</dt>
   <dd>以视觉方式显示信息的装置 ... ...</dd>
</dl>

```



## 浏览器支持

| IE | Firefox | Chrome | Safari | Opera |
| --- | --- | --- | --- | --- |

所有浏览器都支持 &lt;dt&gt; 标签。

## 定义和用法

&lt;dt&gt; 标签定义了定义列表中的项目（即术语部分）。

## HTML 与 XHTML 之间的差异

NONE

## 全局属性

&lt;dt&gt; 标签支持 [HTML 中的全局属性](/tags/html_ref_standardattributes.asp)。

## 事件属性

&lt;dt&gt; 标签支持 [HTML 中的事件属性](/tags/html_ref_eventattributes.asp)。
