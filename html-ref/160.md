# HTML ASCII 参考手册

**HTML 和 XHTML 用标准的 7 比特 ASCII 代码在网络上传输数据。**

**7 比特 ASCII 代码可提供 128 个不同的字符值。**

## 7 比特 设备控制 ASCII代码

ASCII设备控制代码最初被设计为用来控制诸如打印机和磁带驱动器之类的硬件设备。在HTML文档中这些代码不会起任何作用。
